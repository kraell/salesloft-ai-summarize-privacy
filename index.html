  What this extension does                                                                                                                             

  The Salesloft AI Summarizer is an internal tool for Toro TMS employees. It reads activity data visible on Salesloft account and contact pages and uses

   it to generate a structured sales briefing.                                                                                                          

                 

  What data is collected                                                                                                                               

  The extension reads the following information directly from the Salesloft page you are viewing:

  - Account and contact names                                                                                                                          

  - Activity history (calls, emails, notes, meetings)                                                                                                  

  - Profile fields (location, company details, phone numbers)                                                                                          

                                                                                                                                                       

  Where data is sent                                                                                                                                   

  Page data is sent to Anthropic's Claude API solely for the purpose of generating a summary. No data is stored by this extension. No data is sold or  

  shared with any third party.                                                                                                                          

                                                                                                                                                       

  External services used                                                                                                                               

  - Anthropic Claude API — processes activity data to generate summaries

  - Zippopotam.us — converts zip codes to geographic coordinates for proximity calculations                                                            

  - Google Sheets — reads internal Toro TMS reference data (customer list, closed lost records)

                                                                                                                                                       

  Data storage                                                                                                                                         

  This extension does not store personal data. Temporary data may be held in Chrome's local storage solely to cache reference data (customer lists) and

  reduce load times. This cache contains no personal or prospect information beyond what is already in Toro TMS internal records.                      

                 

  Who can use this extension                                                                                                                           

  This extension is for internal use by Toro TMS employees only. It is not publicly distributed.
