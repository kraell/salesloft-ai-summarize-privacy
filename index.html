Privacy Policy — Salesloft AI Summarizer

Last Updated: March 2026

Overview

Salesloft AI Summarizer is a Chrome extension that reads activity data from Salesloft account and contact pages and generates AI-powered sales briefings to assist with outreach.

Data Collected

When the user clicks Summarize, the extension reads from the current Salesloft page:

  • Activity log text (calls, emails, notes, meetings)
  • Account and contact field values (company name, location, truck count, and similar profile fields)

The extension also stores one item persistently in Chrome's sync storage:

  • A Claude API key entered by the user in the extension's settings page

How Data Is Used

Page data is transmitted to Anthropic's Claude API solely to generate a sales briefing for the user. No raw page data is retained by the extension or by Toro TMS after the session ends.

Third-Party Services

  • Anthropic Claude API — receives page activity data to generate the AI summary
  • Zippopotam.us — receives zip codes for geographic distance calculations; no personal data transmitted
  • Google Sheets — retrieves internal reference data only; no Salesloft page data is sent to Google Sheets

Data Sharing

Data is not sold, rented, or shared with any third parties for advertising, analytics, or any purpose beyond those described above.

Data Retention

Activity and profile data is used only during the active session and is not stored after use. The API key is retained in Chrome sync storage until removed by the user via the extension's options page.

User Control

Users can remove their API key at any time via the extension's options page. Uninstalling the extension removes all locally stored data.

Chrome Web Store Limited Use Disclosure

The use of information received from Google APIs will adhere to the Chrome Web Store User Data Policy, including the Limited Use requirements.

Contact

For questions, contact the Toro TMS team.
